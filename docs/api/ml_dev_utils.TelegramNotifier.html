TelegramNotifier¶

class TelegramNotifier[source]¶

Bases: object

Send message to Telegram

This class sends messages to Telegram. To access Telegram, you need to set the environment variables TELEGRAM_BOT_TOKEN and TELEGRAM_CHAT_ID. The documentation on setting up a Telegram bot can be found here: https://core.telegram.org/bots.

Example to set ENV on Mac/Linux:

export TELEGRAM_BOT_TOKEN=your-token-here
export TELEGRAM_CHAT_ID=your-chat-id-here
